# Autonomous AI Research Team

A multi-agent AI system that simulates a research team collaborating to analyze and generate insights from a query.

# Architecture

User Query  
↓  
Research Agent  
↓  
Analysis Agent  
↓  
Critic Agent  
↓  
Planner Agent  
↓  
Final Output  

# Features

- Multi-agent architecture
- Role-based task execution
- Iterative reasoning loop
- Structured output generation
- Streamlit web interface


# Tech Stack

- Python
- Streamlit
- Groq API (LLM-ready)
- Multi-Agent Design
